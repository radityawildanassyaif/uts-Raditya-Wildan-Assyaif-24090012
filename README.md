Deskripsi Project

Project ini adalah implementasi dari desain Figma menjadi website fungsional menggunakan HTML, CSS, dan JavaScript. Website terdiri dari tiga halaman utama:

1. Halaman Login (index.html)
- Input Email & Password  
- Validasi input tidak boleh kosong  
- Jika berhasil → muncul alert “Login berhasil”  
- Redirect menggunakan `window.location.href` ke dashboard  
- Tampilan mengikuti contoh Figma  

2. Halaman Dashboard (dashboard.html)
- Menampilkan header “Dashboard”  
- Terdapat 3 card:
  - Total Produk
  - Total Penjualan
  - Total Revenue
- Data diambil dari array dummy pada `script.js`
- Tombol menuju halaman daftar produk

3. Halaman List Produk (products.html)
- Menampilkan tabel produk menggunakan array of object
- Ditampilkan menggunakan `forEach()`  
- Fitur:
  - **Edit** → alert nama produk
  - **Delete** → konfirmasi lalu menghapus baris tabel dengan DOM (`remove()`)


Teknologi yang Digunakan
- HTML5  
- CSS3  
- JavaScript DOM  
- GitHub Pages Deployment
